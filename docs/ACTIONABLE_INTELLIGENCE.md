# Actionable Intelligence System

Sprint 16 transforms INFOX from "AI summarises news" into "AI helps users understand what to do next." The **Actionable Intelligence** layer adds strategic implications and a watch-entity list to every briefing.

## What It Produces

For each briefing, the action insight engine generates:

### Strategic Implications
Three to five forward-looking bullet points derived from the current news cluster. Examples:

- "If the Thai baht weakens past 37.5 THB/USD, exporters with USD receivables gain. Watch PTT and Charoen Pokphand."
- "Semiconductor supply normalisation reduces input costs for automotive OEMs. Monitor sector EPS revisions Q3."

These are **not summaries of what happened** — they are forward-looking signals that help users decide what to monitor or act on.

### Watch Entities
A curated short list (typically 3–5) of the most important entities to track given the current news cluster. These entities are:

1. **Rising** — mentioned with high acceleration in the last 24h
2. **Strategic** — appear as dominant entities in active narrative threads
3. **Cross-sector** — entities that appear across multiple different news categories

### Urgency Level
Each action insight is tagged with an urgency level:

| Level | Meaning |
|-------|---------|
| `watch` | Monitor — no immediate action required |
| `prepare` | Prepare contingencies — likely development within days |
| `act` | Signal is established — decision window is open |

## Implementation

**Service**: `artifacts/api-server/src/services/intelligence/actionInsight.ts`

The service:
1. Takes the current article cluster and the user's interest profile
2. Queries the entity memory for rising entities related to the cluster
3. Queries the narrative memory for relevant narrative threads
4. Builds a structured prompt for the AI provider using `buildActionInsightPrompt()`
5. Parses the AI response into the structured `ActionInsight` contract

## The Action Insight Contract

```typescript
interface ActionInsight {
  strategicImplications: string[];   // 3–5 forward-looking bullets
  watchEntities: WatchEntity[];      // curated entity list
  urgencyLevel: "watch" | "prepare" | "act";
  timeHorizon: "hours" | "days" | "weeks";
  reasoning: string;                 // brief explanation of the analysis
}

interface WatchEntity {
  entityId: string;
  label: string;
  reason: string;                    // why this entity matters right now
  momentum: "rising" | "stable" | "declining";
}
```

## Integration with Briefing Formatter V3

The action insight block appears as the **fifth section** of the V3 briefing format:

```
1. HEADLINE SIGNAL
2. WHY THIS MATTERS (strategic context)
3. KEY SIGNALS (priority-ranked)
4. STRATEGIC WATCHLIST (action insights)
5. CONFIDENCE TIER
6. READ TIME
```
